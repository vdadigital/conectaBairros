# ConectaBairros v2.0

Projeto acadêmico desenvolvido para a disciplina Sistemas para Internet.

## Tecnologias

- PHP 8.2
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- TailwindCSS
- JavaScript ES6
- HTML5
- GitHub

## Funcionalidades

- Login Google
- Cadastro de negócios
- Pesquisa por categoria
- Pesquisa por estado
- Upload de imagens
- Controle por proprietário
- CRUD completo
